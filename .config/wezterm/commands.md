# WezTerm コマンド一覧

- **Leader**: `Ctrl+q`。押したあと **いったんキーを離してから** 2 秒以内に次のキーを押す（例: `w` でワークスペース一覧）
- **SUPER**: macOS では `Cmd` キー

---

## ワークスペース

| キー | 操作 |
|------|------|
| `Cmd` + `Option` + `W` | ワークスペース一覧（ランチャー）を表示 |
| `Cmd` + `Shift` + `W` | 同上（OS に奪われない環境なら） |
| `Leader` + `w` | ワークスペース一覧を表示して切り替え |
| `Leader` + `$` | 現在のワークスペース名を変更 |
| `Leader` + `Shift` + `W` | 新規ワークスペースを作成して切り替え |
| **Cmd+p** | コマンドパレットからワークスペースの作成・切り替え（一覧UIは出ない） |

---

## コマンドパレット

| キー | 操作 |
|------|------|
| `Cmd` + `p` | コマンドパレットを開く |
| `Ctrl` + `Shift` + `p` | コマンドパレットを開く |

---

## タブ

| キー | 操作 |
|------|------|
| `Ctrl` + `Tab` | 次のタブへ |
| `Ctrl` + `Shift` + `Tab` | 前のタブへ |
| `Leader` + `{` | タブを左に移動 |
| `Leader` + `}` | タブを右に移動 |
| `Cmd` + `t` | 新しいタブを開く |
| `Cmd` + `w` | 現在のタブを閉じる |
| `Cmd` + `1` 〜 `8` | 1〜8番目のタブに切り替え |
| `Cmd` + `9` | 最後のタブに切り替え |

---

## ペイン（分割）

| キー | 操作 |
|------|------|
| `Leader` + `d` | ペインを縦に分割 |
| `Leader` + `r` | ペインを横に分割 |
| `Leader` + `x` | 現在のペインを閉じる |
| `Leader` + `h` | 左のペインにフォーカス |
| `Leader` + `l` | 右のペインにフォーカス |
| `Leader` + `k` | 上のペインにフォーカス |
| `Leader` + `j` | 下のペインにフォーカス |
| `Ctrl` + `Shift` + `[` | ペイン選択モード（矢印で選択） |
| `Leader` + `z` | 現在のペインをズーム（最大化/元に戻す） |

---

## ペインリサイズ（キーテーブル: resize_pane）

| キー | 操作 |
|------|------|
| `Leader` + `s` | リサイズモードに入る |
| `h` / `l` / `k` / `j` | 左 / 右 / 上 / 下にリサイズ |
| `Enter` | リサイズモードを終了 |

---

## ペイン切り替え（キーテーブル: activate_pane）

| キー | 操作 |
|------|------|
| `Leader` + `a` | ペイン切り替えモードに入る（1秒間有効） |
| `h` / `l` / `k` / `j` | 左 / 右 / 上 / 下のペインにフォーカス |

---

## コピー・貼り付け

| キー | 操作 |
|------|------|
| `Leader` + `[` | コピーモードを開始 |
| `Cmd` + `c` | 選択範囲をクリップボードにコピー |
| `Cmd` + `v` | クリップボードから貼り付け |

---

## コピーモード内の操作

### 移動

| キー | 操作 |
|------|------|
| `h` / `j` / `k` / `l` | 左 / 下 / 上 / 右 |
| `0` | 行の左端へ |
| `^` | 行の先頭（空白除く）へ |
| `$` | 行の末尾へ |
| `w` / `b` / `e` | 単語単位で前/後/末尾へ |
| `g` | スクロールバッファの先頭へ |
| `G` | スクロールバッファの末尾へ |
| `H` / `M` / `L` | 画面上端 / 中央 / 下端へ |
| `t` / `f` | 次の文字まで（t: 含む手前、f: 含む） |
| `T` / `F` | 前の文字まで |

### スクロール

| キー | 操作 |
|------|------|
| `Ctrl` + `b` / `f` | ページアップ / ダウン |
| `Ctrl` + `d` / `u` | 半ページ下 / 上 |

### 選択

| キー | 操作 |
|------|------|
| `v` | セル選択モード |
| `Ctrl` + `v` | ブロック選択モード |
| `V` | 行選択モード |

### コピー・終了

| キー | 操作 |
|------|------|
| `y` | 選択範囲をクリップボードにコピー |
| `Enter` | コピーしてコピーモード終了 |
| `Escape` / `Ctrl` + `c` / `q` | コピーモード終了 |

---

## 表示・ウィンドウ

| キー | 操作 |
|------|------|
| `Alt` + `Enter` | フルスクリーン切り替え |

---

## フォント

| キー | 操作 |
|------|------|
| `Ctrl` + `+` | フォントサイズを大きく |
| `Ctrl` + `-` | フォントサイズを小さく |
| `Ctrl` + `0` | フォントサイズをリセット |

---

## その他

| キー | 操作 |
|------|------|
| `Ctrl` + `Shift` + `r` | 設定を再読み込み |
